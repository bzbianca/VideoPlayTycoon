# Video Store Tycoon: The Astra Yao Challenge

A strategic business simulation built in Java. Take on the role of a Video Store Manager in a bustling city. Your goal is to reach **500 Reputation** within **31 Days** to impress the legendary idol, Astra Yao, and secure an invite to her exclusive concert.

---

## 🚀 How to Download and Run

### 1. Project Structure
Ensure your project folder contains the following four Java files and this README:
* `GameEngine.java` (The main controller and entry point)
* `InventoryManager.java` (Logic for stock and shelf organization)
* `Movie.java` (The data model for video tapes)
* `Box.java` (The container for processing shipments)

### 2. Compilation
Open your terminal or command prompt, navigate to the folder containing the files, and run:
```bash
javac *.java
java GameEngine
